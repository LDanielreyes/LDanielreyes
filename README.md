## 🔭 Sobre mí

Soy un **Desarrollador Back-End Junior** especializado en el ecosistema **.NET (C#, Entity Framework Core, SQL)**. Mi trayectoria combina una base sólida en lógica de programación y procesos ETL, con una experiencia previa en gestión administrativa y comercial. Esto me otorga una perspectiva única para entender el flujo de los negocios y traducirlo en soluciones de software eficientes, escalables y seguras.
